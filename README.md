# SupervivenciaFest
Bienvenidos a mi proyecto de Programacion Estatica y Laboratorio Web

Este proyecto se trata de una pagina web sobre festivales musicales como coachella, lollapalooza o la fiesta de la confluencia, etc.

Elegi el genero metal/rock y se llama "Supervivencia Fest", que se hace justo aca en Argentina con re nombres que pasaron por aca.
